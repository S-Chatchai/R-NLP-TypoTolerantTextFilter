# 🎯 Thai Dynamic Fuzzy Matcher & Filter Tool

เครื่องมือสำหรับคัดกรองข้อมูลข้อความภาษาไทยจากไฟล์ Excel (`.xlsx`) โดยใช้หลักการตัดคำภาษาไทยร่วมกับการทำ **Dynamic N-gram** และ **Fuzzy String Matching** ช่วยให้สามารถตรวจจับคำสำคัญ (Keywords) ได้อย่างแม่นยำแม้ว่าคำในข้อมูลดิบจะพิมพ์ผิด (Typo), สะกดเพี้ยน หรือมีคำอื่นมาคั่นกลางก็ตาม

## คุณสมบัติเด่น (Features)

* **Thai NLP Tokenization:** ตัดคำภาษาไทยได้อย่างแม่นยำด้วย Engine `newmm` จากไลบรารี `PyThaiNLP`
* **Dynamic N-gram Generation:** ค้นหาคำแบบยืดหยุ่นรอบคำค้นหา ($n-1, n, n+1$) เพื่อเพิ่มโอกาสการเจอคำที่พิมพ์ตกหรือพิมพ์เกิน
* **Typo-Tolerant Matching:** คำนวณคะแนนความคล้ายด้วย `RapidFuzz` ค้นหาคำผิดได้อย่างมีประสิทธิภาพ
* **Multi-Keyword Set Logic (AND/OR):** * รองรับการจัดกลุ่มคำค้นหาแยกเป็นบรรทัด (แต่ละบรรทัดถือเป็นเงื่อนไขแบบ **OR**)
* ภายในบรรทัดเดียวกันสามารถใช้เครื่องหมายจุลภาค `,` คั่น เพื่อตั้งเงื่อนไขว่าต้องเจอทุกคำในเซตนั้นๆ (เงื่อนไขแบบ **AND**) ถึงจะผ่านเกณฑ์


* **Blacklist Cleansing:** ระบบลบคำที่ไม่ต้องการออกก่อนการตัดคำเพื่อเพิ่มความแม่นยำ (เช่น "การ", "ความ")

## 🚀 การติดตั้งและเตรียมระบบ (Installation)

1. **ติดตั้ง Python** (แนะนำเวอร์ชัน 3.9 ขึ้นไป)
2. **ติดตั้ง Libraries ที่จำเป็น** ผ่าน Terminal หรือ Command Prompt:
```bash
pip install pandas rapidfuzz pythainlp streamlit openpyxl

```


*(หรือสร้างไฟล์ `requirements.txt` แล้วพิมพ์คำสั่ง `pip install -r requirements.txt`)*

## 📖 วิธีการใช้งาน (Usage)

โปรเจกต์นี้ถูกแบ่งออกเป็น 2 รูปแบบตามความสะดวกในการใช้งานของคุณ:

### 1. เวอร์ชัน Terminal / Script (`app_cli.py`)

เหมาะสำหรับสาย Developer หรือต้องการนำไปรันแบบอัตโนมัติ (Automation Task) บนเครื่อง

* **การตั้งค่า:** เปิดไฟล์ `app_cli.py` และแก้ไขตัวแปรโดยตรงที่ด้านล่างของไฟล์ เช่น:
```python
input_file = "CMPC_Monthly_20250531.xlsx"
output_file = "CMPC_Monthly_relevant1.xlsx"
keyword_sets = [
    ["คุณสู้เราช่วย"],
    ["ไถ่ถอน", "โฉนด"]  # ต้องเจอทั้งคำว่า ไถ่ถอน และ โฉนด ในแถวนั้น
]

```


* **วิธีการรัน:**
```bash
python app_cli.py

```


* **ผลลัพธ์:** ระบบจะแสดงความคืบหน้าผ่านหน้าจอ Console และส่งออกไฟล์ Excel ตัวใหม่ตามชื่อที่ระบุใน `output_file` ให้อัตโนมัติ

---

### 2. เวอร์ชัน Web UI ด้วย Streamlit (`app_ui.py`)

เหมาะสำหรับผู้ใช้ทั่วไป (User/Business Analyst) ที่ไม่ถนัดการพิมพ์โค้ด มีหน้าต่าง UI จัดการทุกอย่างได้ง่ายผ่านเว็บเบราว์เซอร์

* **วิธีการรัน:**
```bash
streamlit run app_ui.py

```


* **ขั้นตอนการใช้งานบนหน้าเว็บ:**
1. ไปที่แถบควบคุมด้านซ้าย (**Sidebar**) เพื่อทำการอัปโหลดไฟล์ Excel
2. กำหนดชุดคีย์เวิร์ด และ Blacklist ที่ต้องการกรอง
3. ปรับค่า **Fuzzy Threshold** (คะแนนความเหมือน 0-100) และ **Min Length Ratio** ตามต้องการ
4. กดปุ่ม `▶️ เริ่มประมวลผลคัดกรองข้อมูล`
5. ระบบจะแสดงสถิติ อัตราการจับได้ (Catch Rate) และตาราง Preview ผลลัพธ์ 10 แถวแรกขึ้นมาบนหน้าจอ
6. กดปุ่ม `📥 ดาวน์โหลดไฟล์ Excel ผลลัพธ์` เพื่อเซฟไฟล์ลงเครื่องคอมพิวเตอร์ของคุณ

## ⚙️ คำอธิบายพารามิเตอร์ที่สำคัญ

* **Fuzzy Threshold (ค่าเริ่มต้น: 80):** ระดับความใจดีในการดักจับคำผิด ยิ่งค่าน้อยยิ่งจับคำที่พิมพ์เพี้ยนไปมากๆ ได้ (แต่หากตั้งน้อยเกินไปอาจเกิด False Positive หรือจับคำที่ไม่เกี่ยวมาด้วย)
* **Min Length Ratio (ค่าเริ่มต้น: 0.7):** สัดส่วนความยาวของคำที่ดักจับได้เทียบกับคีย์เวิร์ดหลัก เพื่อป้องกันไม่ให้คำสั้นๆ ไปแมตช์มั่วกับคำที่ยาวกว่ามาก
